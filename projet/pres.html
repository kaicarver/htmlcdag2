Présentation en 4 parties :

  1. Contexte - à quel besoin répond l’appli ?
  2. Cible - à qui s’adresse l’appli ?
  3. Concurrence - quels sont les 3 principaux concurrents de l’appli ?
  4. Fonctionnalités

Contexte

  JournalDeBord (nom à definir) est une application qui permet de tenir un journal de bord personnel.
  Comme le journal de bord d’un capitaine de bateau.
  
  Exemple de discussion de ce type d'application :
  Comment garder un journal de bord
  
  Je n'ai pas trouvé d'application exactement de ce type.

Cible et acteurs

Cible

  L'application est destinée à des personnes qui veulent garder une trace de leurs activités et qui écrivent :
  
  soignant
  secrétaire
  chercheur
  compositeur
  écrivain
  explorateur
  navigateur de bateau

Acteurs

  Les acteurs sont :
    les personnes qui écrivent
      qui peuvent etre catégorisés en différents types, 
    l'administrateur du site
      qui gère le site, les différents types, et répond aux questions

Concurrence

Types de logiciels similaires :
  
  Logiciel de blog: Blogspot, Wordpress, Movable Type
  Traitement de texte : Google Docs, Microsoft Word, Etherpad/Framapad
  Outil d’écriture : 750words.com, telegra.ph
  Réseau social : Twitter/X, Facebook
  Appli de prise de notes : Microsoft OneNote, Apple Notes, Obsidian
  Messagerie : Telegram, WhatsApp, Gmail
  Éditeur de texte : VS Code, Vim, Emacs
  
  https://zapier.com/blog/best-note-taking-apps/
  https://telegra.ph/Test-de-Telegraph-10-22

Fonctionnalités

de base :

Prendre des notes textuelles rapidement.
Fonctionne sur tous appareils : desktop, mobile, tablette.
Données enregistrées dans le cloud.
Organise les notes par date.
Faire des recherches parmi les notes.
Publier une note de différentes façons.
Comptage des mots.
Statistiques sur l’activité en fonction du temps.
Sécurité et confidentialité.

avancées (à faire quand on a le temps) :

Gère l’historique des modifications de chaque note.
Facilite l’écriture de textes ou phrases similaires.
Rapports de différents types.
Mise en forme avec Markdown ou autre.
Monitoring de tendances.
Intégration avec des photos.
Intégration avec données environnementales : localisation, type d’appareil, pouls, vitesse, …
Sécurité et confidentialité avancées : personne ne peut lire les données sauf l’utilisateur 
Utilisation en groupe.
IA : résumé, recherche par thème, …
