Mes documents en cours d'écriture

Pour l'examen oral du diplôme de Concepteur développeur d'applications (CDA), il faut soumettre plusieurs documents au jury :

un dossier projet, un dossier professionnel, et un support de présentation.

Voici sur quoi je travaille :
  • Dossier projet :
    • PDF : dossier-projet.pdf
    • HTML : dossier-projet.html
    générés par make(1) à partir du fichier Markdown : dossier-projet.md
    autres, vieilles versions : dynamic, vieux from slides
  • Dossier professionnel :
    • dernière version : doc Word officiel renseigné (éditable en ligne sur OneDrive)
    • versions en local :
      • doc Word (non éditable sans license Office)
      • version PDF (par impression PDF du doc en ligne)
    • Vieilles version locales : 1 2
  • Support de présentation 2

Sujets abordés dans le cours

  • présentation projet full stack
  • installer et configurer environnement de travail en fonction du projet
  • HTML/CSS : bases et avancé
  • conception des applications Web : Merise, UML
  • JavaScript client : bases, jQuery, jQuery UI, plugins
  • JavaScript serveur : Node.js, framework Express.js
  • NoSQL : MongoDB
  • JavaScript client : React
  • PHP sur serveur
  • SQL : MySQL (via WampServer), interaction avec PHP
  • concevoir et mettre en place une base de données relationnelle
  • développer des composants d'accès de données SQL et NoSQL
  • responsive design et Bootstrap
  • prototypages UX/UI, design
  • analyse de besoins, maquettage application
  • interfaces utilisateur
  • gestion de versions décentralisée : git
  • atelier CV
  • Devops, mise en production (Docker ? Kubernetes ?)
  • veille technique et concurrentielle sur Internet
  • architecture de l'information, ergonomie, livrables, UI/UX, tendances du design
  • architecture logicielle d'une application
  • contribuer à la gestion de projet informatique
  • gestion de projet : Agile et Scrum, WBS, planning, budgetisation
  • PHP avancé, programmation orientée objet
  • framework Symfony
  • cybersécurité, cryptographie, sécurité application Web
